Detailed Description

Main class that wraps interactions with inference API. choose_rank and report_outcome make up the two logical steps in the inference API. The relevant C++ class is reinforcement_learning::live_model.

Choose an action, given a list of actions, action features and context features. The inference library chooses an action by creating a probability distribution over the actions and then sampling from it.

Parameters
context_jsonContains action, action features and context features in JSON format
event_idThe unique identifier for this interaction. This event_id must be used when reporting the outcome for this action
deferredIdentifies if action is deferred or not. Deferred one requires additional call of report_action_taken to be involved in training.
Exceptions
RuntimeErrorThrown if an internal error occurs
Returns
model_id, chosen_action_id, actions_probabilities
model_id - Get the model id. Every call to choose action is associated with a unique model used to predict. A unique model id is associated with each unique model. (This is set internally by the API)
chosen_action_id - ID of action chosen by model
actions_probabilities - List of pairs (action_id, probability) ranked by the model's prediction
